### 基于ECC加密的传输模型
<hr>

#### 说明

##### 前言
	*  该项目是帮老师开发的一款软件
	*  也是我开发的一个功能还算完善的C#软件
* 基于ECC椭圆曲线加密，的信息传递。
	* ECC加密为非对称加密
	* 数据加密，数据验证
	* 公私钥存储，分发
* 对ECC加密算法重构
* 系统还考虑到客户端的合法性，采用CPU序列号+硬盘序列号的联合验证
* 用到了Scoket技术，
* C/S架构，支持多用户同时登陆。server端具备实时查看在线用户
